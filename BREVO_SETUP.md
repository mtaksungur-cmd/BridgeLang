# Brevo (Sendinblue) Email Setup Guide

Once you receive your Brevo API Key, follow these steps to enable emails:

## 1. Update `.env.local`
Add or update the following variables:

```env
# Brevo SMTP Settings
SMTP_HOST=smtp-relay.brevo.com
SMTP_PORT=587
SMTP_USER=your_brevo_account_email@example.com
SMTP_PASS=your_brevo_api_key_here
MAIL_FROM="BridgeLang <contact@bridgelang.co.uk>"
```

## 2. Test the connection
You can trigger a test email by registering a new student or using the "Forgot Password" feature.

## 3. Verify Sender in Brevo
Ensure that `contact@bridgelang.co.uk` is a verified sender in your Brevo dashboard under **Settings > Senders & IP**.
